5 <2020-04-05 Sun 08:22> update

I finally find a way around Nvidia driver issue and installed Ubuntu on this machine. I am so excited. A quick note of the solution:

5.1

  1. Boot to safe graphics mode (a mode in case of graphics card driver is not available)
  2. Go to Additional drivers. Change to a closer server. (sohu's server gave me an error. I used Tsinghua's server.)
  3. Open a terminal. sudo apt-add-repository ppa:graphics-drivers/ppa && sudo apt update. I selected Nvidia package under additional drivers. Then install as normal and reboot.

5.2 The issue is it reboots to black screen.

  1. Press Esc key to grub interface when rebooting.
  2. Clt+Alt+F6 to open a terminal
  3. sudo apt pure nvidia
  4. ubuntu-drivers devices check info. It gives me recommended driver name ("nvidia-driver-440").
  5. nvidia-smi check info. it is used as a measure of whether Nvidia driver is installed successfully.
  6. sudo reboot

6 <2020-04-05 Sun 19:22> update: chinese input on ubuntu

  1. install chinese input on ubuntu store
  2. open input method
  3. run ibus-setup and add chinese input method called intelligence pinyin
  4. run ibus restart if chinese input is not available under input method tab
  5. open settings->region&languages, add input source named intelligence pinyin
