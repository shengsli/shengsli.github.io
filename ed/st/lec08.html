1 Modified condition/decision (MC/DC)

  • industry requirements: boolean testing shall satisfy MC/DC
  • motivation: effectively test important conbinations of conditions
    • important combination means that each basic condition shown to independently affect the outcome of each decision
  • requires:
    • basic condition C, 2 test cases
    • values of all evaluated conditions except C are the same
    • compound condition as a whole evaluates to true for one and false for the other

1.1 Remarks

  1. each atomic condition: true or false
  2. evaluate all conditions
  3. each atomic condition independently affect the outcome
