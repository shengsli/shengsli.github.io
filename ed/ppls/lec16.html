1 Linda

  • Linda is an alternative parallelism model.
  • An implementation of Linda is JavaSpace.
  • key concept: tuple space
    • Tuple space is a global content addressable memory.
    • shared and accessible by all processes
    • cf. database
    • content addressable
    • no names, have addresses
    • talk to them by their content
      • e.g. ask for tuples that has "green" value in some fields
    • atomic operations:
      • create new tuples: no half created tuples
      • ask for tuples in select-style, cf. database
