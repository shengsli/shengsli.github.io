6 simple nested loops join

for each tuple r belongto R:	outer table
  for each tuple s belongto S:  inner table
    if (r == s) emit (r,s)

6.1 example database:

  • R: M pages, m tuples
  • S: N pages, n tuples
  • M = 1000, m = 100,000
  • N = 500, n = 40,000
  • cost analysis:
    • 0.1 ms per I/O
    • \(M + m \cdot N\)
    • \(N + n \cdot M\)
  • remarks: let smaller relation be the outer relation

7 block nested loops join

  • block = page 
for each block b_r belongto R:		outer table
  for each block b_S belongto S: 	inner table
    for each tuple r belongto b_R: 	
      for each tuple s belongto b_S:
        emit if r and s match
  • cost = \(N + M \cdot N\) = 500,500 I/Os -> 50 sec
  • cost = \(M + M \cdot N\)
  • cost = \(M + (\frac{M}{B-2} \cdot N)\)

8 block nested loops join: B buffers

for each B-2 block b_R belongto R:
  for each block b_S belongto S:
    for each tuple r belongto b_R:
      for each tuple s belongto b_S:
        emit if r and s match
  • cost = \(M + \frac{M}{B-2} \cdot N\)
