1 buffer manager

  • buffer manager brings pages from disk to memory and back
  • manages buffer pool
  • buffer pool is a designated main memory area for pages.
    • it is in-memory cache of disk pages.
  • higher-level code can
    - pin request a page from the buffer manager
    - unpin release a page
  • metadata per frame
    dirty flag indicate if the page is modified
    pin counter # of concurrent users of the page
      if pin counter = 0, the page is candidate for replacement
